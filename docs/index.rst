Welcome to EdFoil's documentation!
===================================

**EdFoil** (/ˈɛd.fɔɪl/) is a Python library for the design of wind and tidal blades based on airfoils.
It pulls data from various sources and offers a *simple* and *intuitive* API.

It also comes as an installable executable for Windows.

Check out the :doc:`usage` section for further information, including
how to :ref:`installation` the project.

.. note::

   This project is under active development.
